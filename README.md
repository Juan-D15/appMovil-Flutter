\# Task Management App



Aplicación móvil desarrollada con Flutter para la gestión de tareas. Permite a los usuarios crear, visualizar, actualizar y eliminar tareas de manera sencilla desde un dispositivo móvil.



\## Descripción



La aplicación está diseñada para facilitar la organización de actividades diarias mediante un sistema simple de gestión de tareas. Los usuarios pueden registrar tareas, consultar su lista de pendientes y administrar su progreso.



\## Características



\* Creación de tareas

\* Visualización de lista de tareas

\* Actualización de tareas existentes

\* Eliminación de tareas

\* Interfaz sencilla y fácil de usar



\## Tecnologías utilizadas



\* Flutter

\* Dart

\* Material Design



\## Estructura del proyecto



```text

lib/

&#x20;├── main.dart

&#x20;├── screens/

&#x20;├── widgets/

&#x20;├── models/

&#x20;└── services/



assets/

```



\* `main.dart`: punto de entrada de la aplicación

\* `screens`: contiene las pantallas principales de la aplicación

\* `widgets`: componentes reutilizables de la interfaz

\* `models`: modelos de datos utilizados en la aplicación

\* `services`: lógica de negocio o manejo de datos



\## Requisitos



\* Flutter SDK

\* Dart SDK

\* Android Studio, VS Code u otro editor compatible

\* Un emulador o dispositivo físico



\## Instalación



Clonar el repositorio:



```bash

git clone https://github.com/usuario/repositorio.git

```



Entrar al directorio del proyecto:



```bash

cd repositorio

```



Instalar dependencias:



```bash

flutter pub get

```



\## Ejecución



Para ejecutar la aplicación:



```bash

flutter run

```



\## Compilación



Generar APK para Android:



```bash

flutter build apk

```



Generar versión web:



```bash

flutter build web

```



\## Pruebas
